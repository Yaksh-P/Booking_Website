# Database Files

This project runs with JSON files in `database/data` for easy local execution, and it also includes relational SQL scripts for assignment submission.

- `schema.sql` defines the tables for users, services, time slots, and bookings.
- `seed.sql` provides starter records that mirror the default JSON seed data.
